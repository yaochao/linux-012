# `rebuild/userland/src/`

[中文 README](./README.md)

This directory stores the source files for the minimal userland programs.

## Current Files

- `crt0.S`
  process startup entry
- `lib.c`
  lightweight runtime helpers and syscall wrappers
- `sh.c`
  minimal shell implementation
- `ls.c`
  minimal `ls` command implementation
