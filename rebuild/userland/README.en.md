# `rebuild/userland/`

[中文 README](./README.md)

This directory stores the repo-owned minimal Linux 0.12 userland source and linker configuration. It is the source of the userland needed to enter a shell and run commands.

## Main Contents

- `include/`
  header files
- `src/`
  source files for the minimal userland programs
- `linker.ld`
  linker script
